# Workplace Safety Survey Analysis

This project analyzes survey data to evaluate workplace safety awareness, PPE usage, accident rates, and regulatory compliance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Visualization

## Project Overview

The dataset consists of survey responses collected to assess safety practices and awareness levels within a workplace environment.

## Key Metrics Analyzed

- Safety Awareness Rate
- PPE Usage Rate
- Accident Occurrence Rate
- Regulation Awareness

## Key Insights

- Safety awareness varies across respondents.
- PPE usage is not consistent across all participants.
- Some respondents reported workplace accidents.
- There are gaps in regulatory awareness that may require attention.
